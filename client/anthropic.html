How you can use Claude

Learning

Learn anything through conversation. Upload docs or images1 for deeper insights or create visual study aids with artifacts2. Voice3 makes complex topics easier to digest, while projects4 keeps your learning organized.

Coding

Build anything with intelligent help. Debug issues5, learn new languages, optimize algorithms with clear explanations6, or analyze existing code7.

Researching

Gather information from everywhere—search the web8, connect Google Workspace9, use custom integrations10—and create interactive reports with reliable citations11 in artifacts.

Analyzing

Make sense of anything complex. Analyze data12, create visualizations13, review documents, evaluate decisions, or break down key choices.

Creating

Turn ideas into reality. Brainstorm and refine concepts, then bring them to life—from polished writing14 to interactive artifacts15.

Tooltips

  1. Uploading docs & images
    Upload any file (PDF, Word, Excel) or image and Claude analyzes it to answer your questions. 
  2. Artifacts
    Turn ideas into shareable creations. Build tools, visualizations, and experiences by simply describing what you need.
  3. Voice mode
    Switch between typing and speaking to Claude. Perfect for when you’re on the move, or want to think out loud.
  4. Projects
    Organize conversations by topic with persistent context. Keep related work together and build on previous insights.
  5. Code generation and debugging
    Write and fix code in any language. Claude explains errors and suggests solutions to accelerate your development.
  6. Step-by-step explanations
    Complex concepts broken down simply. Learn programming, math, or science with clear explanations tailored to your level.
  7. Code analysis
    Review existing code with suggestions for optimization and best practices.
  8. Web search
    Access current information beyond Claude’s knowledge. Find recent data, news, and research to supplement your work.
  9. Google Workspace
    Search and pull context from your Gmail, calendar, and docs for a complete picture.
  10. Integrations
    Connect Claude to your favorite tools like JIRA, Zapier, and Intercom, or build your own.
  11. Research artifacts with citations
    Build shareable reports with proper citations. Choose from APA, MLA, Chicago, and other citation styles.
  12. Data analysis
    Upload files to identify patterns, calculate stats, and get actionable insights for work or personal decisions.
  13. Data visualization
    Create charts and diagrams to make data clear. Share insights with visual aids that highlight key patterns.
  14. Writing and content creation
    Draft, edit, and polish any kind of writing. Claude helps maintain your voice while improving clarity and structure.
  15. Interactive artifacts
    Build interactive content: documents, visualizations, code, and diagrams. Share what you create with anyone, even if they don’t use Claude.
